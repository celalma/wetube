# WeTube
WeTube is a website/web application which offers a better video content consumption experience. Build from the YouTube database and thanks to [Youtube's API](https://developers.google.com/youtube/v3?hl=en).
In other words, WeTube has all the content and videos that YouTube can offer.

## Why it's better
Using useful extensions created by [SharkPool](https://sharkpools-extensions.vercel.app/?tag=all), it allows to play videos in the app ***without any ads*** adn without **video suggestions** meaning complete privacy and assuring more focus by getting rid of the distracting suggestions.

## But how to use it?
It's very simple. Dowload the *WeTube.html ( this file is not yet present in th repositiory. It will be added udring beta)* file and execute it **or** download the WeTube.sb3 file and run it.
